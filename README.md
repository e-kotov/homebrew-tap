# Egor Kotov's Homebrew Tap

This repository contains Homebrew formulas for various tools and applications developed by Egor Kotov.

## How to use

To add this tap to your Homebrew installation, run:

```bash
brew tap e-kotov/tap
```

Then you can install formulas using:

```bash
brew install <formula-name>
```

Alternatively, you can install a tool directly without tapping the repository first:

```bash
brew install e-kotov/tap/<formula-name>
```
